# Install

## via Composer

composer.json
```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/andreaskasper/php_lib_cryptocom"
        }
    ],
    "require": {
        "andreaskasper/cryptocom": "*"
    },
    "minimum-stability": "dev"
}
```

then use composer for example via docker

```console
foo@bar:~$ docker run -it --rm -v ${PWD}:/app/ composer update
```
